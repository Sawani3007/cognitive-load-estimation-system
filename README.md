# Cognitive Load Detection System

A real-time computer vision system that estimates cognitive load using
eye blink patterns captured via webcam.

## Features
- Real-time face & eye tracking (MediaPipe)
- Blink detection using Eye Aspect Ratio (EAR)
- Cognitive load estimation
- Data logging & visualization
- ML-ready pipeline

## Tech Stack
- Python
- OpenCV
- MediaPipe
- NumPy, Pandas
- Streamlit
